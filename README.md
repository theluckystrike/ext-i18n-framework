# ext-i18n-framework

Internationalization framework for Chrome extensions.

## Installation

```bash
npm install @zovo/ext-i18n-framework
```

## Usage

```typescript
import ext_i18n_framework from '@zovo/ext-i18n-framework';

const i18n = new ext_i18n_framework();
```

## API

### ext_i18n_framework

The main class for the i18n framework.

```typescript
new ext_i18n_framework()
```
